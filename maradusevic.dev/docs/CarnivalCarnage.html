Carnival Carnage

Carnival Carnage is a Virtual Reality Arcade game where you must dash around an arena, using a mallet to destroy clown-headed robots. The game is set within a giant big top tent with a dark and mysterious atmosphere.

While at AIE, we were able to develop a VR game in Unity. After being placed in a team, we created Carnival Carnage, a fun arcade game where the player smashes these creepy moving clown-heads. Everytime the player smashes one, depending on the type, the player adds more time to the scoreboard. More time left means the player scores more points, getting players to aim for faster runs everytime.

Hammer Movement & Scoring System

For the game, I created the physics behind the hammer's movement as well as the scoring system. Also the UI elements for the scoreboard, win and lose screens and the small time additions that appear after killing an enemy.
